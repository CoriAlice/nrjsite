Petites explications et consignes pour utiliser correctement notre site :

-Démarrer la navigation su la page d'accueil (management/accueil).
-Une bar de menu permet d'acceder aux autres pages.
-Vous pouvez vous inscrire sur la page d'accueil (votre mot de passe sera enregisté de manière sécurisée dans la base de donnée).
-Seulement accueil et inscription sont ouverts au public, il faut être connecté pour accéder aux pages "Les sites", "Les voies" et "Carte".

Nous avons choisi l'option D - Affichage sur une carte et l'option E - Affichage de statistiques sur un site 
Mais nous avons fait en plus la gestion des identifications (option B). 

-la page "Les sites" affiche la liste de tous les sites. 
Vous pouvez afficher les détails d'un site en cliquant sur la "loupe" ou le supprimer en cliquant sur la "poubelle".
Vous pouvez ajouter un nouveau site (pensez à mettre des corrdonées GPS réelles pour une meilleure simulation)

-la page details site vous permet d'accéder à toutes les informations concernant le site mais aussi 
d'ajouter un nouveau relevé
d'ajouter une nouvelle voie d'acheminement (seulement les sites de type inverse sont proposés - vous pouvez raccorder un site producteur qu'à des sites consommateurs et vice-versa)
d'éditer le site

-la page "Carte" affiche les sites localisés sur la carte. 
Sur les balises un indice "C" ou "P" permet d'indiquer si c'est un site Consommateur ou Producteur.
Vous pouvez cliquer sur les balises pour afficher les informations du site correspondant.
